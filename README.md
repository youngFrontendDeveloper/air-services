#Air Services - Дополнительные услуги для авиапассажиров

Веб-приложение для просмотра и управления дополнительными услугами авиакомпаний. Разработано на React с использованием Vite.

### Установка

```bash
# Клонировать репозиторий
git clone https://github.com/youngFrontendDeveloper/air-services.git
cd air-services

# Установить зависимости
npm install
# или
yarn install
```

### Запуск в режиме разработки

```bash
npm run dev
# или
yarn dev
```

Приложение будет доступно по адресу: [http://localhost:5173](http://localhost:5173)

### Сборка для production

```bash
npm run build
# или
yarn build
```

Собранные файлы будут помещены в папку `dist/`

### Просмотр собранной версии

```bash
npm run preview
# или
yarn preview
```

**Живая версия:** [https://youngFrontendDeveloper.github.io/air-services/](https://youngFrontendDeveloper.github.io/air-services/)

```

## Возможности

### Управление услугами
- Просмотр категорий дополнительных услуг
- Добавление услуг из заказа
- Расчет общей стоимости

### Пользовательский интерфейс
- Адаптивный дизайн (Mobile-first)
