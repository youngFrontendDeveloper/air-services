Air Services - Дополнительные услуги для авиапассажиров

Веб-приложение для просмотра и управления дополнительными услугами авиакомпаний. Разработано на React с использованием Vite.

### Что требуется:
1. Страница со списком услуг (данные можно захардкодить или замокать).
2. Для каждой услуги показать:
- название
- цену
- кнопку «Добавить»
3. При нажатии на кнопку услуга добавляется в список выбранных.
4. Отобразить блок «Итого»:
- выбранные услуги
- общую сумму
- кнопку «Оформить заказ» (без логики оплаты)
5. Интерфейс должен быть адаптивным (mobile-first).

Технические требования:
- React + TypeScript
- функциональные компоненты
- корректная работа со состоянием
- аккуратная и понятная структура кода

### Установка

```bash
 Клонировать репозиторий
git clone https://github.com/youngFrontendDeveloper/air-services.git
cd air-services

 Установить зависимости
npm install
 или
yarn install
```

### Запуск в режиме разработки

```bash
npm run dev
 или
yarn dev
```

Приложение будет доступно по адресу: [http://localhost:5173](http://localhost:5173)

### Сборка для production

```bash
npm run build
 или
yarn build
```

Собранные файлы будут помещены в папку `dist/`

### Просмотр собранной версии

```bash
npm run preview
 или
yarn preview
```
